Introduction

JavaScript ek programming language hai jo web pages ko interactive banata hai.

Ye browser me directly chalti hai aur HTML ke sath use hoti hai.

JavaScript se hum popups, sliders, forms, games, sab kuch bana sakte hain.

  • Client-side language
  • Runs in browser

Syntax

Syntax matlab hota hai JavaScript likhne ke rules.

Statements ko semicolon (;) se end karte hain, aur code blocks curly braces {} me likhte hain.

// Example
let name = "Loveneesh";
console.log(name);
  • Semicolon optional but recommended
  • Code blocks use { }

Variables

Variable ek box jaisa hota hai jisme data store hota hai.

JavaScript me hum let, const, ya var se variable banate hain.

let age = 25;
const name = "Ravi";
var city = "Delhi";
  • let = block scope
  • const = constant value

Functions

Function ek block hota hai jo koi kaam karta hai.

Hum function ko call karke use baar baar use kar sakte hain.

function greet(name) {
return "Hello " + name;
}

greet("Loveneesh");
  • Functions are reusable blocks

Loops

Loop ka use tab hota hai jab hume koi kaam baar baar repeat karna ho.

JavaScript me for, while jaise loop hote hain.

for (let i = 0; i < 5; i++) {
console.log("Count: " + i);
}
  • for loop is most used
